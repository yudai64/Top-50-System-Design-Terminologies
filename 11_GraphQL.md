# GraphQL
- https://www.digitalocean.com/community/tutorials/an-introduction-to-graphql

## What I learned
- GraphQL is an open-source query language and runtime for APIs. GraphQL was invented by developers at Facebook to solve various issues encountered with traditional REST APIs.
- GraphQL Characteristics
  - Declarative
  - Hierarchical
  - Strongly-typed
  - Self-documenting
  - Client-driven
